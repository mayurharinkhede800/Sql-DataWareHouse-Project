# Data Warehouse Development & ETL Implementation
### Tools: SQL Server, T-SQL, SSMS, VS Code
Designed and implemented a multi-layer Data Warehouse architecture (Bronze,
Silver, Gold) using SQL Server schemas.
Ingested raw CRM source files into the Bronze layer using structured T-SQL scripts.
Built Silver layer tables by performing data cleansing, type conversions,
deduplication, and standardization.
Implemented TRUNCATE & LOAD ETL strategy to ensure consistent and fresh data
loads.
Created reusable T-SQL scripts for loading multiple tables with consistent naming
conventions.
Validated data accuracy using row-count checks and reconciliation queries.
